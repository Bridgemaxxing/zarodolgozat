# 2D Dark Fantasy Turn-Based Game

Team-based game development project focused on turn-based combat systems, progression mechanics, and database-driven gameplay.

## Overview

A dark fantasy inspired turn-based combat game where players progress through encounters, quests, and boss battles while developing their character through stat growth and equipment systems.

## My Role — Gameplay / Systems Development

I worked primarily on gameplay systems and combat implementation. My responsibilities included:

Designing and implementing the turn-based combat flow
Building player stat and leveling frameworks
Implementing enemy scaling and encounter balance
Integrating MySQL / MariaDB database structures
Developing quest battle logic and progression loops
Supporting React-based combat UI integration

## Tech Stack

JavaScript
React
Node.js
MySQL / MariaDB
Godot Engine

## Team Project Context

This project was developed in a 3-person development team as part of a software development program. I collaborated closely with teammates responsible for frontend structure and visual production.
